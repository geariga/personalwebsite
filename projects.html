NASA Near Earth Object Data Visualizer

I began this project after discovering how fun public APIs can be. So far, it allows the user to enter a date (or a range of dates), which is used to extract information from the API so the user can view data about Near-Earth-Objects for the given date(s). This may include the number of NEOs, how many were potentially hazardous, and also their proximity, size, and the next estimated perihelion. I intend to use visual aids to give a more user-friendly view of the data.

  • Concepts Used:
  • Public APIs
  • Data extraction and visualization
  • Front-end (HTML5, CSS3, JS)
  • Asynchronous functionality
